# 🚗 MyRating.autos — Бездушный алгоритм оценки авто

Инструмент для тех, кому нужны цифры, а не эмоции. Этот сервис анализирует данные об автомобилях и выставляет беспристрастный рейтинг на основе математических формул.

## 🧠 Как это работает?
Алгоритм оценивает машину по 100-балльной шкале, учитывая:
* **Возраст:** Автоматический «штраф» за каждый год эксплуатации.
* **Пробег:** Влияние износа на итоговую стоимость.
* **Рыночная цена:** Сравнение текущего ценника со средним по рынку.
* **Ликвидность:** Поправочные коэффициенты для разных марок.

## 🛠 Технологии
* **HTML5 / CSS3** (Адаптивный дизайн для комфортной работы с ноутбука и телефона)
* **JavaScript (Vanilla)** — вся логика и расчеты без лишних библиотек.
* **VS Code** — основная среда разработки.
* **Vercel** — хостинг и CI/CD.

# MyRating.autos🚗💨

**Беспристрастный алгоритм оценки рыночной стоимости и инвестиционной привлекательности автомобилей.**

Проект создан для автоматизации поиска выгодных предложений на вторичном рынке. В основе лежит "бездушный алгоритм", который отсекает эмоции и оценивает машину по жестким техническим и экономическим критериям.

### 🛠 Технологии
* **Frontend:** HTML5, CSS3 (адаптивная верстка под мобильные устройства)
* **Logic:** Pure JavaScript (ES6+)
* **Deployment:** Vercel / GitHub Pages
